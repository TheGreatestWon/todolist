**문서 버전:** 1.0

**기반 문서:** PRD(2-PRD.md), 사용자 시나리오(3-user-scenario.md), 도메인 정의서(1-domain-definition.md)

## 1. 와이어프레임 개요

이 문서는 개인용 할 일 관리 시스템의 사용자 인터페이스 와이어프레임을 정의합니다. 사용자 시나리오, 기능 요구사항, 도메인 개념을 바탕으로 직관적이고 효율적인 인터페이스를 설계합니다.

## 4. 컴포넌트 명세

### 4.1 헤더 컴포넌트
- 애플리케이션 로고/이름
- 사용자 이메일 표시
- 로그아웃 버튼

### 4.2 할 일 추가 폼 컴포넌트
- 제목 입력 필드
- 설명 입력 필드 (접기/펼치기 가능)
- 마감일 선택기
- 추가 버튼

### 4.3 할 일 카드 컴포넌트
- 완료 체크박스
- 제목 텍스트
- 설명 텍스트 (존재할 경우)
- 마감일 표시
- 편집/삭제 버튼
- 상태별 스타일 (완료, 기한 경과 등)

### 4.4 섹션 컴포넌트

- 카운트 표시

## 7. 사용자 흐름

1. **회원가입/로그인**
   - 로그인 페이지 → 인증 → 할 일 목록 페이지

2. **할 일 추가**
   - 할 일 목록 페이지 → 할 일 추가 폼 → 할 일 생성

3. **할 일 조회 및 관리**
   - 할 일 목록 페이지 → 섹션별 표시 → 할 일 상호작용

4. **할 일 수정/삭제**
   - 할 일 목록 → 편집 버튼 → 수정 모달 → 저장/취소

5. **할 일 완료 처리**
   - 할 일 목록 → 체크박스 클릭 → 상태 변경 및 이동

6. **로그아웃**
   - 헤더 → 로그아웃 버튼 → 로그인 페이지
